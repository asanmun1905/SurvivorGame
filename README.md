Una migración moderna y optimizada del clásico juego de supervivencia en consola a una experiencia web inmersiva de alto rendimiento.

---

## ✨ Características Principales

- **🎩 Estética Premium**: Interfaz moderna con efectos de _glassmorphism_, gradientes vibrantes y fondos animados.
- **⚡ Motor Canvas**: Renderizado fluido utilizando la API de Canvas 2D con efectos de iluminación (_Bloom_) en tiempo real.
- **🧠 IA Inteligente**: Algoritmos de movimiento basados en distancias Manhattan y Euclídea para persecuciones y huidas realistas.
- **🛠️ Totalmente Customizable**: Panel de control lateral para ajustar el tamaño del tablero, cantidad de entidades y velocidad de simulación.
- **🏗️ Arquitectura Limpia**: Código escrito en TypeScript siguiendo principios de programación orientada a objetos (POO) y patrones de diseño.

## 🚀 Cómo Empezar

Para ejecutar el proyecto localmente, asegúrate de tener [Node.js](https://nodejs.org/) instalado y sigue estos pasos:

1.  **Instala las dependencias**:

    ```bash
    npm install
    ```

2.  **Inicia el servidor de desarrollo**:

    ```bash
    npm run dev
    ```

3.  **Abre tu navegador**:
    Visita `http://localhost:5173` para empezar a jugar.

---

## 📂 Estructura del Proyecto

El código está organizado de forma modular para facilitar su mantenimiento:

- `src/engine/`: Core del juego, gestión del tablero y ciclo de vida.
- `src/entities/`: Definición de personajes (Enemigo, Presa, Obstáculo).
- `src/logic/`: Estrategias de movimiento e inteligencia artificial.
- `src/utils/`: Utilidades matemáticas y manejo de posiciones.
- `src/style.css`: Estilos visuales de última generación.

## 🛠️ Tecnologías Utilizadas

- **TypeScript**: Para un desarrollo seguro y estructurado.
